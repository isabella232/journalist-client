# Journalist UI

The journalist gem sidekick.

## Installation

Assuming you have node.js and `npm` installed, type:

```shell
npm install -g grunt grunt-cli
npm install
```

And then execute:

    $ grunt

This will run the UI and any existing profiles will pop up on the left.
